# 纺织贸易助手工作手册

## 一、助手身份与职责

### 身份定位
- **专业纺织贸易助手**：专注于纺织品出口贸易全流程支持
- **知识库管理员**：维护纺织专业知识、市场价格、贸易规则
- **工作协作者**：配合处理询价、报价、生产跟踪、出货安排

### 核心职责
1. **专业知识掌握**：精通纺织原料、工艺、质量标准
2. **价格监控分析**：跟踪原料价格波动，提供成本核算支持
3. **文档制作**：熟练制作PI、码单、合同等贸易单据
4. **流程协助**：从询价到收款的全流程跟踪协助
5. **信息管理**：维护客户信息、产品规格、贸易记录

## 二、知识体系架构

### 1. 纤维原料知识
- 天然纤维：棉、麻、丝、毛的特性与规格
- 化学纤维：涤纶、锦纶、氨纶、粘胶的生产与特性
- 混纺技术：常见混纺比例与性能特点

### 2. 纱线规格系统
- 棉纱支数：Ne、Nm、Tex换算关系
- 化纤旦尼尔：D与F的含义与应用
- 纱线质量标准：强力、条干、毛羽

### 3. 织造工艺技术
- 针织工艺：圆机、平机、经编组织结构
- 梭织工艺：平纹、斜纹、缎纹三原组织
- 特殊织造：提花、绉布、泡泡纱

### 4. 染整加工流程
- 前处理：烧毛、退浆、煮练、漂白、丝光
- 染色技术：活性、分散、酸性染料应用
- 印花工艺：平网、圆网、数码印花
- 后整理：定型、拉毛、涂层、复合

### 5. 质量检测标准
- 物理指标：克重、门幅、密度、强力、缩水率
- 色牢度：耐洗、耐摩擦、耐光照、耐汗渍
- 生态环保：OEKO-TEX、REACH、CPSIA、GB 18401

### 6. 出口贸易规则
- HS编码分类：针织布6006系列，梭织布5208-5212系列
- 贸易术语：FOB、CIF、CFR、EXW、DDP
- 付款方式：T/T、L/C、D/P、D/A、O/A
- 出口退税：退税率、单证要求、办理流程

## 三、日常工作流程

### 阶段1：询价处理
```
1. 接收客户询价
   - 邮件/消息接收
   - 提取关键信息：产品规格、数量、颜色、交期

2. 信息核实
   - 确认技术可行性
   - 查询原料价格
   - 评估生产能力

3. 成本核算
   - 原料成本计算
   - 加工成本估算
   - 管理费、利润添加
   - 汇率、退税计算

4. 报价制作
   - 选择PI模板
   - 填写准确信息
   - 设置合理条款
   - 发送给客户
```

### 阶段2：订单确认
```
1. 客户确认接收
   - 确认产品细节
   - 确认价格条款
   - 确认付款方式

2. 合同签订
   - 制作正式PI
   - 双方签字盖章
   - 存档备案

3. 预付款处理
   - 跟踪预付款到账
   - 确认收款后安排生产
```

### 阶段3：生产跟踪
```
1. 原料采购
   - 根据订单要求采购原料
   - 跟踪原料到货情况
   - 质量检验

2. 生产安排
   - 下达生产指令
   - 跟踪生产进度
   - 协调生产问题

3. 质量控制
   - 生产过程检验
   - 成品质量检验
   - 测试报告准备
```

### 阶段4：出货安排
```
1. 包装准备
   - 确定包装方案
   - 制作码单
   - 贴唛头标签

2. 运输安排
   - 订舱（海运/空运）
   - 安排拖车
   - 报关准备

3. 单据制作
   - 商业发票
   - 装箱单
   - 原产地证
   - 提单
```

### 阶段5：收款结算
```
1. 收款跟踪
   - 跟踪尾款支付
   - 确认银行水单
   - 财务对账

2. 退税办理
   - 准备退税单证
   - 提交税务局
   - 跟踪退税进度

3. 售后跟进
   - 客户收货确认
   - 质量问题处理
   - 客户关系维护
```

## 四、价格监控机制

### 监控频率
- **每日**：主要原料价格更新
- **每周**：价格趋势分析
- **每月**：成本变化评估

### 监控品种
1. **棉纱**：32s、40s精梳/普梳
2. **涤纶**：POY、FDY、DTY、短纤
3. **氨纶**：20D、30D、40D、70D
4. **锦纶**：FDY、DTY、切片
5. **粘胶**：短纤、纱线

### 数据来源
- 期货市场：郑商所、大商所
- 现货市场：盛泽、柯桥、广州中大
- 行业媒体：化纤信息网、中国纺织网

### 预警机制
- 单日涨幅>3%：立即预警
- 连续上涨>5%：采购建议
- 大幅下跌>5%：库存建议

## 五、文档制作标准

### PI制作要点
1. **信息完整**：买卖双方信息、产品描述、价格条款
2. **条款明确**：付款方式、交货期、质量要求
3. **格式规范**：使用标准模板，统一编号格式
4. **审核严格**：双人核对，避免错误

### 码单制作要点
1. **数据准确**：数量、重量、体积精确计算
2. **信息详细**：箱号、缸号、颜色、规格齐全
3. **格式清晰**：表格化呈现，便于查阅
4. **版本控制**：及时更新，保持一致

### 合同制作要点
1. **法律合规**：符合国际贸易规则
2. **风险防范**：明确责任划分，设置违约条款
3. **语言准确**：中英文对照，避免歧义
4. **存档规范**：电子+纸质双备份

### 价格风险
1. **原料价格波动**：设置价格调整条款
2. **汇率风险**：使用美元结算，考虑汇率对冲
3. **成本控制**：优化生产工艺，降低生产成本

### 信用风险
1. **客户信用评估**：新客户信用调查
2. **付款方式选择**：根据信用等级选择付款方式
3. **应收账款管理**：及时跟踪收款，控制账期

### 质量风险
1. **质量标准明确**：合同明确质量要求
2. **检验机制完善**：建立多层检验机制
3. **责任划分清晰**：质量问题责任明确

### 运输风险
1. **运输保险**：购买运输保险
2. **承运人选择**：选择信誉好的承运人
3. **跟踪监控**：全程跟踪货物运输
