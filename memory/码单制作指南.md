# 码单（Packing List）制作指南

## 一、码单基本概念

### 定义
码单是详细记录货物包装情况的单据，包括箱号、数量、重量、体积等信息，是报关、运输、收货的重要文件。

### 作用
1. **报关**：海关查验货物依据
2. **运输**：船公司/航空公司配载依据
3. **收货**：客户清点货物依据
4. **结算**：运费计算依据

## 二、码单基本要素

### 必须包含的信息
1. **卖方信息**：公司名称、地址、联系方式
2. **买方信息**：客户公司名称、地址
3. **发票号**：对应的商业发票号码
4. **日期**：制作码单的日期
5. **运输信息**：船名/航班号、提单号
6. **包装详情**：箱号、产品描述、数量、重量、体积
7. **总计**：总箱数、总毛重、总净重、总体积
8. **唛头**：运输标志

## 四、重量计算规则

### 1. 净重计算
```
标准公式：N.W. = G.W. - 包装材料重量

常用标准：
- 纸箱包装：G.W. - 0.4kg
- 编织袋包装：G.W. - 0.2kg
- 托盘包装：G.W. - 托盘重量(15-25kg)
```

### 2. 体积计算
```
标准纸箱：55×35×35cm = 0.067 CBM
计算公式：CBM = 长(m) × 宽(m) × 高(m)

常见规格：
- 小箱：40×30×30cm = 0.036 CBM
- 中箱：55×35×35cm = 0.067 CBM  
- 大箱：60×40×40cm = 0.096 CBM
```

### 3. 面料重量换算
```
针织布重量计算：
1. 已知克重：重量(kg) = 长度(m) × 门幅(m) × 克重(gsm) ÷ 1000
2. 已知码重：重量(kg) = 码数 × 码重(kg/yd) × 0.9144

示例：
- 1000码，门幅64"，克重125gsm
- 计算：1000yd × 0.9144 × (64"×0.0254) × 125 ÷ 1000 = 186.5kg
```

## 五、缸号批号管理系统

### 1. 缸号编码规则
```
格式：字母 + 年月日 + 3位序列号
示例：A260309163

解释：
- A：生产线代码（A线、B线、C线）
- 260309：2026年3月9日
- 163：当日第163缸
```

### 2. 批号管理要点
1. **同一缸号**：保证颜色一致性
2. **生产记录**：记录染色配方、工艺参数
3. **质量追溯**：出现问题可追溯具体缸号
4. **客户要求**：部分客户要求提供缸号清单

### 3. 码单中的缸号记录
```
对于大货生产：
- 按缸号分批包装
- 同一缸号集中装箱
- 码单中注明每个箱子的缸号
- 提供缸号汇总表给客户
```

## 六、包装规格标准

### 1. 卷装规格
```
标准卷装：
- 每卷长度：50码、100码、150码
- 卷径：30-35cm
- 卷芯：纸质或塑料
- 内包装：塑料薄膜
- 外包装：编织袋或纸箱

特殊要求：
- 防水包装：加防潮纸
- 防污包装：加防尘罩
- 易碎品：加强包装
```

### 2. 箱装规格
```
标准纸箱：
- 材质：5层瓦楞纸
- 尺寸：55×35×35cm（标准出口箱）
- 承重：≤25kg
- 封箱：胶带十字封箱
- 标识：印刷或贴标

装箱规则：
- 20卷/箱（50码卷）
- 10卷/箱（100码卷）
- 避免过度挤压
- 均匀分布重量
```

### 3. 托盘规格
```
标准托盘：
- 尺寸：110×110cm（标准）
- 材质：木质或塑料
- 高度：≤150cm（含货物）
- 重量：≤1000kg
- 缠绕：塑料薄膜缠绕

堆码规则：
- 40箱/托盘（标准箱）
- 交错堆码增加稳定性
- 四角加护角
- 顶部加盖板
```

## 九、海关申报要点

### 1. 商品归类
```
关键信息：
- HS编码：准确归类
- 商品描述：与HS编码匹配
- 成分比例：影响归类
- 加工工艺：影响归类

常见问题：
- 针织vs梭织归类不同
- 混纺面料按主要成分
- 特殊功能面料特殊归类
```

### 2. 原产地申报
```
原产地标准：
- 完全获得：中国生产
- 实质性改变：中国加工
- 区域价值成分：满足百分比

文件准备：
- 原产地证（CO）
- 加工工序说明
- 原料来源证明
```

### 3. 数量申报
```
计量单位：
- 主要单位：千克（kg）
- 辅助单位：米（m）/码（yd）
- 换算关系：准确换算

允许误差：
- 一般±3%
- 特殊要求±1%
- 实际数量为准
```

## 十一、电子码单系统

### 1. 数据格式
```
标准字段：
- CartonNo：箱号
- ProductCode：产品代码
- ColorCode：颜色代码
- LotNo：缸号
- Quantity：数量
- Unit：单位
- GrossWeight：毛重
- NetWeight：净重
- Length：长
- Width：宽
- Height：高
- Volume：体积
```

### 2. 文件格式
```
常用格式：
- Excel：.xlsx，便于编辑
- PDF：.pdf，不可修改
- CSV：.csv，系统对接
- XML：.xml，数据交换

选择原则：
- 客户要求优先
- 报关要求其次
- 内部系统兼容
```

## 十二、工作流程

### 1. 准备阶段
```
步骤：
1. 收集生产完成数据
2. 获取实际测量数据
3. 整理缸号批号信息
4. 确定包装方案
5. 准备模板文件
```

### 2. 制作阶段
```
步骤：
1. 填写基础信息
2. 录入详细数据
3. 计算汇总统计
4. 核对数据准确性
5. 生成最终版本
```

### 3. 审核阶段
```
审核要点：
1. 数据一致性检查
2. 计算准确性验证
3. 格式规范性审查
4. 特殊要求满足
5. 签字盖章完整
```

### 4. 分发阶段
```
分发对象：
1. 报关行（报关用）
2. 货代（订舱用）
3. 客户（清关用）
4. 内部存档（记录用）
5. 财务部门（结算用）
```

## 十三、常见问题处理

### 1. 数据不一致
```
处理方案：
1. 重新测量核实
2. 查找错误原因
3. 更正错误数据
4. 记录更正过程
5. 通知相关方
```

### 2. 客户特殊要求
```
应对策略：
1. 仔细阅读要求
2. 评估实施难度
3. 沟通确认细节
4. 调整制作方案
5. 记录特殊处理
```

### 3. 紧急修改
```
应急流程：
1. 评估修改影响
2. 快速实施修改
3. 通知所有相关方
4. 更新所有版本
5. 记录修改历史
```
