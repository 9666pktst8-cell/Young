## 七、价格预警机制

### 预警阈值
- **大幅上涨**：单日涨幅>3%
- **大幅下跌**：单日跌幅>3%
- **持续上涨**：连续5日累计涨幅>5%
- **持续下跌**：连续5日累计跌幅>5%

### 预警响应
1. 立即通知相关采购/销售部门
2. 调整报价策略
3. 评估库存风险
4. 制定应对方案

## 八、成本计算模型

### 面料成本构成
```
原料成本 = 纱线价格 × 单耗
织造成本 = 织造费 + 损耗
染整成本 = 染色费 + 后整理费
管理成本 = 人工 + 水电 + 折旧
利润 = 总成本 × 利润率
含税价 = (总成本 + 利润) × (1 + 增值税率)
美元报价 = 含税价 ÷ 汇率 ÷ (1 - 退税率)
```

### 计算示例（针织布）
- 原料：32s棉纱，价格29,500元/吨
- 单耗：1.2kg/米
- 织造费：3元/米
- 染色费：5元/米
- 管理费：2元/米
- 利润率：10%
- 增值税：13%
- 退税率：13%
- 汇率：7.2
