# 纺织专业知识速查手册

## 一、纤维知识库

### 天然纤维
#### 棉（Cotton）
- **长绒棉**：埃及棉（Giza）、美国Pima棉、新疆长绒棉
- **细绒棉**：陆地棉，占全球产量90%
- **特性**：吸湿性好（8%）、透气、柔软、易皱、缩水

#### 麻（Linen）
- **亚麻**：凉爽、吸湿快干、强度高、易皱
- **苎麻**：中国特产，有"中国草"之称
- **特性**：抑菌、抗紫外线、挺括

#### 丝（Silk）
- **桑蚕丝**：家蚕，品质最好
- **柞蚕丝**：野蚕，色泽较黄
- **特性**：光泽好、滑爽、吸湿、娇贵

#### 毛（Wool）
- **羊毛**：美利奴羊毛最细
- **羊绒**：克什米尔山羊，纤维直径14-16μm
- **特性**：保暖、弹性好、缩绒性

### 化学纤维
#### 涤纶（Polyester）
- **原料**：PTA + MEG → PET
- **品种**：FDY（全拉伸丝）、DTY（低弹丝）、POY（预取向丝）
- **特性**：强度高、抗皱、易洗快干、吸湿差

#### 锦纶（Nylon）
- **尼龙6**：己内酰胺聚合
- **尼龙66**：己二酸+己二胺聚合
- **特性**：耐磨性最好、弹性好、易静电

#### 氨纶（Spandex）
- **莱卡**：杜邦公司注册商标
- **特性**：弹性极好（拉伸5-8倍）、强度低、不耐氯

#### 粘胶（Viscose）
- **原料**：木浆、棉浆
- **品种**：粘胶短纤、长丝、Modal、Tencel
- **特性**：吸湿好（13%）、悬垂好、湿强低

## 二、纱线规格系统

### 棉纱支数
- **英制支数（Ne）**：1磅纱线有多少个840码
  - 20s：中等厚度
  - 32s：较细
  - 40s：细纱
  - 60s：高支纱

- **公制支数（Nm）**：1kg纱线有多少个1000米
- **特克斯（Tex）**：1000米纱线的重量（克）

### 化纤旦尼尔
- **旦尼尔（D）**：9000米纤维的重量（克）
  - 75D：轻薄面料
  - 150D：中等厚度
  - 300D：厚重面料

- **单丝根数（F）**：一束丝中的单丝数量
  - 72F：较细
  - 144F：较粗
  - 288F：很粗

### 混纺表示
- **T/C**：涤棉混纺，如65/35
- **T/R**：涤粘混纺
- **CVC**：棉为主（棉含量>50%）的涤棉混纺

## 三、织造工艺详解

### 针织（Knitting）
#### 纬编（Weft Knitting）
- **圆机**：汗布、珠地、卫衣布、罗纹
- **平机**：横机，织片状
- **组织结构**：
  - 平纹：最基本的单面组织
  - 罗纹：1×1、2×2，弹性好
  - 双罗纹：双面布，不易卷边
  - 提花：图案复杂
  - 网眼：透气性好

#### 经编（Warp Knitting）
- **特里科经编**：弹性好，内衣、泳衣
- **拉舍尔经编**：绒类织物、窗帘
- **特点**：不易脱散、尺寸稳定

### 梭织（Weaving）
#### 三原组织
1. **平纹（Plain）**
   - 经纬纱一上一下
   - 最紧密，最结实
   - 府绸、帆布、牛津布

2. **斜纹（Twill）**
   - 经纬纱至少三上一下
   - 有斜向纹路
   - 卡其、牛仔布、华达呢

3. **缎纹（Satin）**
   - 经纬纱至少五上一下
   - 表面光滑，光泽好
   - 贡缎、丝绸

#### 特殊组织
- **提花（Jacquard）**：复杂图案
- **绉布（Crepe）**：表面有绉效应
- **泡泡纱（Seersucker）**：有泡泡效果

## 四、染整工艺全流程

### 前处理（Pretreatment）
1. **烧毛（Singeing）**
   - 去除表面绒毛
   - 提高光洁度

2. **退浆（Desizing）**
   - 去除织造浆料
   - 淀粉酶、氧化剂退浆

3. **煮练（Scouring）**
   - 去除天然杂质（蜡质、果胶）
   - 碱煮练

4. **漂白（Bleaching）**
   - 提高白度
   - 双氧水漂白

5. **丝光（Mercerizing）**
   - 浓碱处理
   - 增加光泽、强力

### 染色（Dyeing）
#### 染料类型
1. **活性染料（Reactive）**
   - 用于棉、麻、粘胶
   - 色牢度好
   - 需要固色

2. **分散染料（Disperse）**
   - 用于涤纶
   - 高温高压染色
   - 热熔染色

3. **酸性染料（Acid）**
   - 用于羊毛、丝绸、锦纶
   - 色牢度中等

4. **直接染料（Direct）**
   - 用于棉、粘胶
   - 工艺简单，色牢度差

#### 染色方法
- **浸染（Exhaust）**：面料浸在染液中
- **轧染（Pad）**：面料通过染液轧辊
- **卷染（Jig）**：面料在染缸中往复运动

### 印花（Printing）
#### 印花工艺
1. **平网印花**
   - 适合小批量、复杂图案
   - 套色精度高

2. **圆网印花**
   - 适合大批量
   - 速度快

3. **数码印花**
   - 无版印花
   - 适合小批量、个性化

4. **转移印花**
   - 主要用于涤纶
   - 图案精细

#### 印花浆料
- **涂料印花**：工艺简单，手感较硬
- **活性印花**：色牢度好，手感柔软
- **分散印花**：用于化纤面料

## 五、后整理工艺大全

### 物理整理
1. **定型（Setting）**
   - 控制门幅、克重
   - 控制缩水率

2. **拉毛/磨毛（Raising/Sanding）**
   - 表面起绒
   - 增加保暖性

3. **压光（Calendering）**
   - 增加光泽
   - 提高平整度

4. **预缩（Pre-shrinking）**
   - 控制缩水率
   - 桑福赖预缩

### 化学整理
1. **柔软整理（Softening）**
   - 硅油柔软剂
   - 氨基硅油

2. **防水整理（Waterproofing）**
   - C6防水剂（环保）
   - C8防水剂（效果好）

3. **阻燃整理（Flame Retardant）**
   - 磷系阻燃剂
   - 氮系阻燃剂

4. **抗菌整理（Antibacterial）**
   - 银离子抗菌
   - 季铵盐抗菌

### 功能整理
1. **涂层（Coating）**
   - PU涂层：聚氨酯
   - PVC涂层：聚氯乙烯
   - TPU涂层：热塑性聚氨酯

2. **复合（Laminating）**
   - TPU膜复合
   - PTFE膜复合（防水透气）
   - 透气膜复合

3. **贴合（Bonding）**
   - 绒布贴合
   - 海绵贴合
   - 无纺布贴合

## 六、质量检测标准

### 物理指标
1. **克重（GSM）**
   - 轻薄：<100gsm
   - 中等：100-200gsm  
   - 厚重：>200gsm

2. **门幅（Width）**
   - 常见：44/45"、58/60"、62/64"、71/73"
   - 公差：±1英寸

3. **密度（Density）**
   - 针织：横密、纵密（针/英寸）
   - 梭织：经密、纬密（根/英寸）

4. **强力（Strength）**
   - 拉伸强力
   - 撕裂强力
   - 顶破强力

5. **缩水率（Shrinkage）**
   - 经向：≤3%
   - 纬向：≤5%

### 色牢度标准
1. **耐洗色牢度（Washing）**
   - ISO 105-C06
   - 要求：3-4级

2. **耐摩擦色牢度（Rubbing）**
   - ISO 105-X12
   - 干磨：3-4级，湿磨：2-3级

3. **耐光照色牢度（Light）**
   - ISO 105-B02
   - 要求：3-4级

4. **耐汗渍色牢度（Perspiration）**
   - ISO 105-E04
   - 要求：3-4级

### 生态环保标准
1. **OEKO-TEX Standard 100**
   - 有害物质检测
   - 产品级别：I-IV级

2. **REACH法规**
   - SVHC高关注物质
   - 目前219项

3. **CPSIA（美国）**
   - 铅含量限制
   - 邻苯二甲酸酯限制

4. **GB 18401（中国）**
   - 国家强制标准
   - A、B、C三类

## 七、面料成本计算模型

### 原料成本计算
```
1. 纱线单耗计算
   针织布：单耗(kg/m) = 克重(gsm) × 门幅(m) ÷ 1000
   梭织布：考虑经纬纱密度和支数

2. 纱线成本
   原料成本 = 纱线单价(元/kg) × 单耗(kg/m)

3. 染化料成本
   染色费：根据颜色深浅、工艺复杂程度
   助剂费：柔软剂、固色剂等
```

### 加工成本计算
```
1. 织造成本
   针织：圆机费、平机费
   梭织：织机费

2. 染整成本
   前处理费：烧毛、退浆、煮练、漂白
   染色费：根据染料类型、工艺
   后整理费：定型、拉毛、压光等

3. 管理成本
   人工费：工人工资
   水电费：生产能耗
   折旧费：设备折旧
```

### 报价计算
```
1. 成本汇总
   总成本 = 原料成本 + 加工成本 + 管理成本

2. 利润添加
   含利润价 = 总成本 × (1 + 利润率)

3. 含税计算
   含税价 = 含利润价 × (1 + 增值税率)

4. 美元报价
   美元报价 = 含税价 ÷ 汇率 ÷ (1 - 退税率)
```

## 八、常见面料品种

### 针织面料
1. **汗布（Single Jersey）**
   - 最基本针织布
   - 用途：T恤、内衣

2. **珠地（Pique）**
   - 表面有凹凸效果
   - 用途：Polo衫

3. **卫衣布（Fleece）**
   - 内里起绒
   - 用途：卫衣、运动服

4. **罗纹（Rib）**
   - 弹性好
   - 用途：领口、袖口、下摆

5. **网眼布（Mesh）**
   - 透气性好
   - 用途：运动服、鞋材

### 梭织面料
1. **府绸（Poplin）**
   - 平纹，经密>纬密
   - 用途：衬衫

2. **牛津布（Oxford）**
   - 平纹，双经单纬
   - 用途：衬衫、箱包

3. **牛仔布（Denim）**
   - 斜纹，靛蓝染色
   - 用途：牛仔裤

4. **卡其布（Khaki）**
   - 斜纹，卡其色
   - 用途：裤子、工装

5. **贡缎（Sateen）**
   - 缎纹，光泽好
   - 用途：高档服装

## 十、工作配合要点

### 询价阶段
1. **信息收集**
   - 客户具体要求
   - 产品规格参数
   - 数量、颜色、交期

2. **成本核算**
   - 查询原料价格
   - 计算加工成本
   - 考虑汇率、退税

3. **报价制作**
   - 选择合适模板
   - 填写准确信息
   - 设置合理条款

### 生产阶段
1. **进度跟踪**
   - 原料采购进度
   - 织造生产进度
   - 染整加工进度

2. **质量控制**
   - 生产过程检验
   - 成品质量检验
   - 测试报告准备

3. **包装出货**
   - 包装方案确认
   - 码单制作
   - 订舱安排

### 售后阶段
1. **单据处理**
   - 商业发票制作
   - 报关单据准备
   - 清关文件提供

2. **收款跟踪**
   - 付款进度跟踪
   - 银行水单确认
   - 财务对账

3. **客户维护**
   - 发货通知
   - 到货确认
   - 问题处理
