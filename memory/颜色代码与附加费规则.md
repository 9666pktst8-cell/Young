## 六、价格计算规则

### 单价确定因素
1. **面料类型**：网眼布、平纹布、梭织布价格不同
2. **克重**：克重越高，单价可能越高或越低（取决于工艺）
3. **门幅**：门幅影响生产效率，影响价格
4. **颜色**：特殊颜色可能有附加费
5. **数量**：批量大小影响单价

### 总价计算公式
```
总价 = (数量 × 单价) + 附加费

示例：
数量：575.0 kg
单价：2.55 USD/kg
附加费：0.0 USD
总价：575.0 × 2.55 + 0.0 = 1466.25 USD
```

### 附加费计算规则
1. **固定附加费**：150 USD（常见）
2. **比例附加费**：按比例计算（较少见）
3. **条件附加费**：满足特定条件时收取

## 七、工作应用指南

### 制作PI时的颜色处理
1. **检查颜色代码**：确认颜色是否有标准代码
2. **判断附加费**：根据规则判断是否需要附加费
3. **说明附加费**：在PI中明确标注附加费金额和原因
4. **客户沟通**：提前与客户确认特殊颜色附加费

### 价格报价策略
1. **基础价格**：根据面料类型、克重、门幅确定
2. **颜色调整**：考虑颜色工艺难度
3. **数量调整**：考虑生产经济性
4. **附加费说明**：透明化附加费用

### 客户沟通要点
1. **颜色确认**：提供色卡或样品确认颜色
2. **附加费说明**：解释附加费的原因（特殊染料、工艺等）
3. **价格有效期**：明确价格有效期
4. **条款确认**：确认付款、装运等条款
